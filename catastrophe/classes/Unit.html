Methods

Unit:attack (tile)
Attacks an adjacent Tile. Costs an action for each Unit in this Unit's squad. Units in the squad without an action don't participate in combat. Units in combat cannot move afterwards. Attacking structures will not give materials.

Parameters:

  • tile Tile The Tile to attack.

Returns:

    bool True if successfully attacked, false otherwise.
Unit:changeJob (job)
Changes this Unit's Job. Must be at max energy (100) to change Jobs.

Parameters:

  • job string The name of the Job to change to.

Returns:

    bool True if successfully changed Jobs, false otherwise.
Unit:construct (tile, type)
Constructs a Structure on an adjacent Tile.

Parameters:

  • tile Tile The Tile to construct the Structure on. It must have enough materials on it for a Structure to be constructed.
  • type string The type of Structure to construct on that Tile.

Returns:

    bool True if successfully constructed a structure, false otherwise.
Unit:convert (tile)
Converts an adjacent Unit to your side.

Parameters:

  • tile Tile The Tile with the Unit to convert.

Returns:

    bool True if successfully converted, false otherwise.
Unit:deconstruct (tile)
Removes materials from an adjacent Tile's Structure. You cannot deconstruct friendly structures (see `Unit.attack`).

Parameters:

  • tile Tile The Tile to deconstruct. It must have a Structure on it.

Returns:

    bool True if successfully deconstructed, false otherwise.
Unit:drop (tile, resource[, amount=0])
Drops some of the given resource on or adjacent to the Unit's Tile. Does not count as an action.

Parameters:

  • tile Tile The Tile to drop materials/food on.
  • resource string The type of resource to drop ('materials' or 'food').
  • amount number The amount of the resource to drop. Amounts <= 0 will drop as much as possible. (default 0)

Returns:

    bool True if successfully dropped the resource, false otherwise.
Unit:harvest (tile)
Harvests the food on an adjacent Tile.

Parameters:

  • tile Tile The Tile you want to harvest.

Returns:

    bool True if successfully harvested, false otherwise.
Unit:move (tile)
Moves this Unit from its current Tile to an adjacent Tile.

Parameters:

  • tile Tile The Tile this Unit should move to.

Returns:

    bool True if it moved, false otherwise.
Unit:pickup (tile, resource[, amount=0])
Picks up some materials or food on or adjacent to the Unit's Tile. Does not count as an action.

Parameters:

  • tile Tile The Tile to pickup materials/food from.
  • resource string The type of resource to pickup ('materials' or 'food').
  • amount number The amount of the resource to pickup. Amounts <= 0 will pickup as much as possible. (default 0)

Returns:

    bool True if successfully picked up a resource, false otherwise.
Unit:rest ()
Regenerates energy. Must be in range of a friendly shelter to rest. Costs an action. Units cannot move after resting.

Returns:

    bool True if successfully rested, false otherwise.
Unit:log (message)
(inherited) Adds a message to this GameObject's logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters:

  • message string A string to add to this GameObject's log. Intended for debugging.
