Class Game

Convert as many humans to as you can to survive in this post-apocalyptic wasteland.

Fields

self.name The name of this game, "{game_name}".
self.catEnergyMult The multiplier for the amount of energy regenerated when resting in a shelter with the cat overlord.
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.harvestCooldown The amount of turns it takes for a Tile that was just harvested to grow food again.
self.jobs All the Jobs that Units can have in the game.
self.lowerHarvestAmount The amount that the harvest rate is lowered each season.
self.mapHeight The number of Tiles in the map along the y (vertical) axis.
self.mapWidth The number of Tiles in the map along the x (horizontal) axis.
self.maxTurns The maximum number of turns before the game will automatically end.
self.monumentCostMult The multiplier for the cost of actions when performing them in range of a monument.
self.monumentMaterials The number of materials in a monument.
self.neutralMaterials The number of materials in a neutral Structure.
self.players List of all the players in the game.
self.session A unique identifier for the game instance that is being played.
self.shelterMaterials The number of materials in a shelter.
self.startingFood The amount of food Players start with.
self.starvingEnergyMult The multiplier for the amount of energy regenerated when resting while starving.
self.structures Every Structure in the game.
self.tiles All the tiles in the map, stored in Row-major order.
self.timeAddedPerTurn The amount of time (in nano-seconds) added after each player performs a turn.
self.turnsBetweenHarvests After a food tile is harvested, the number of turns before it can be harvested again.
self.turnsToCreateHuman The number of turns between fresh humans being spawned on the road.
self.turnsToLowerHarvest The number of turns before the harvest rate is lowered (length of each season basically).
self.units Every Unit in the game.
self.wallMaterials The number of materials in a wall.

Fields

self.name
The name of this game, "{game_name}".
self.catEnergyMult
The multiplier for the amount of energy regenerated when resting in a shelter with the cat overlord.
self.currentPlayer
The player whose turn it is currently. That player can send commands. Other players cannot.
self.currentTurn
The current turn number, starting at 0 for the first player's turn.
self.gameObjects
A mapping of every game object's ID to the actual game object. Primarily used by the server and client to easily refer to the game objects via ID.
self.harvestCooldown
The amount of turns it takes for a Tile that was just harvested to grow food again.
self.jobs
All the Jobs that Units can have in the game.
self.lowerHarvestAmount
The amount that the harvest rate is lowered each season.
self.mapHeight
The number of Tiles in the map along the y (vertical) axis.
self.mapWidth
The number of Tiles in the map along the x (horizontal) axis.
self.maxTurns
The maximum number of turns before the game will automatically end.
self.monumentCostMult
The multiplier for the cost of actions when performing them in range of a monument. Does not effect pickup cost.
self.monumentMaterials
The number of materials in a monument.
self.neutralMaterials
The number of materials in a neutral Structure.
self.players
List of all the players in the game.
self.session
A unique identifier for the game instance that is being played.
self.shelterMaterials
The number of materials in a shelter.
self.startingFood
The amount of food Players start with.
self.starvingEnergyMult
The multiplier for the amount of energy regenerated when resting while starving.
self.structures
Every Structure in the game.
self.tiles
All the tiles in the map, stored in Row-major order. Use `x + y * mapWidth` to access the correct index.
self.timeAddedPerTurn
The amount of time (in nano-seconds) added after each player performs a turn.
self.turnsBetweenHarvests
After a food tile is harvested, the number of turns before it can be harvested again.
self.turnsToCreateHuman
The number of turns between fresh humans being spawned on the road.
self.turnsToLowerHarvest
The number of turns before the harvest rate is lowered (length of each season basically).
self.units
Every Unit in the game.
self.wallMaterials
The number of materials in a wall.
