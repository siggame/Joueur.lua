Class Game

The simple version of American Checkers.

An 8x8 board with 12 checkers on each side that must move diagonally to the opposing side until kinged.

Fields

self.name The name of this game, "{game_name}".
self.boardHeight The height of the board for the Y component of a checker.
self.boardWidth The width of the board for X component of a checker.
self.checkerMoved The checker that last moved and must be moved because only one checker can move during each players turn.
self.checkerMovedJumped If the last checker that moved jumped, meaning it can move again.
self.checkers All the checkers currently in the game.
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.maxTurns The maximum number of turns before the game will automatically end.
self.players List of all the players in the game.
self.session A unique identifier for the game instance that is being played.
self.timeAddedPerTurn The amount of time (in nano-seconds) added after each player performs a turn.
